💾 Harddisk কী?

💽 Hard Disk (Hard Disk Drive - HDD)

Hard Disk বা HDD হলো কম্পিউটারের একটি স্থায়ী (Non-Volatile) Storage Device, যেখানে Operating System, Software, Documents, Photos, Videos এবং অন্যান্য ফাইল সংরক্ষণ করা হয়।

কম্পিউটার বন্ধ হয়ে গেলেও Hard Disk-এর তথ্য মুছে যায় না।

💡 সহজ ভাষায়: Hard Disk হলো কম্পিউটারের একটি বড় আলমারির মতো, যেখানে সব গুরুত্বপূর্ণ ফাইল দীর্ঘ সময় ধরে নিরাপদে সংরক্ষণ করা হয়।

⚙️ Hard Disk-এর কাজ

  • Operating System সংরক্ষণ করে।
  • ছবি, ভিডিও, গান ও ডকুমেন্ট সংরক্ষণ করে।
  • সফটওয়্যার ও গেম ইনস্টল করে রাখে।
  • ব্যবহারকারীর তথ্য দীর্ঘদিন নিরাপদে সংরক্ষণ করে।
  • কম্পিউটার চালুর জন্য প্রয়োজনীয় ফাইল সংরক্ষণ করে।
  • Backup Data সংরক্ষণে সাহায্য করে।
  • বড় আকারের ফাইল সংরক্ষণে ব্যবহৃত হয়।

⭐ Hard Disk-এর বৈশিষ্ট্য

  • Non-Volatile Storage Device।
  • বড় Storage Capacity (500GB, 1TB, 2TB বা তার বেশি)।
  • চৌম্বকীয় (Magnetic) ডিস্ক ব্যবহার করে।
  • SSD-এর তুলনায় ধীরগতির।
  • দীর্ঘমেয়াদে তথ্য সংরক্ষণ করে।
  • Desktop ও Laptop-এ ব্যবহৃত হয়।

📊 Hard Disk-এর প্রকারভেদ

প্রকার বর্ণনা
Internal HDD কম্পিউটারের ভিতরে স্থায়ীভাবে লাগানো থাকে।
External HDD USB-এর মাধ্যমে বাইরে থেকে সংযুক্ত করা যায়।
Portable HDD ছোট আকারের, সহজে বহনযোগ্য।
Enterprise HDD Server ও Data Center-এ ব্যবহৃত হয়।

✅ Hard Disk-এর সুবিধা

  • 📂 অনেক বেশি পরিমাণ ডেটা সংরক্ষণ করা যায়।
  • 💰 SSD-এর তুলনায় দাম কম।
  • 📸 ছবি, ভিডিও, গান ও ডকুমেন্ট দীর্ঘদিন সংরক্ষণ করা যায়।
  • 💾 Backup নেওয়ার জন্য খুবই উপযোগী।
  • 🖥 Desktop ও Laptop—উভয় ক্ষেত্রেই ব্যবহার করা যায়।
  • 📦 500GB, 1TB, 2TB, 4TB এমনকি 10TB-এর বেশি স্টোরেজ পাওয়া যায়।

❌ Hard Disk-এর অসুবিধা

  • 🐢 SSD-এর তুলনায় ধীরগতির।
  • ⚙️ ভিতরে চলমান যন্ত্রাংশ (Moving Parts) থাকায় সহজে ক্ষতিগ্রস্ত হতে পারে।
  • 🔊 কাজ করার সময় শব্দ হতে পারে।
  • ⚡ বেশি বিদ্যুৎ ব্যবহার করে।
  • 💥 পড়ে গেলে বা আঘাত পেলে ডেটা নষ্ট হওয়ার ঝুঁকি থাকে।

📜 Hard Disk-এর ইতিহাস

১৯৫৬ সালে IBM বিশ্বের প্রথম Hard Disk Drive তৈরি করে। এর স্টোরেজ ছিল মাত্র 5 MB, কিন্তু আকার ছিল একটি বড় আলমারির মতো।

বর্তমানে 1TB, 2TB, 4TB, 8TB এমনকি 20TB-এরও বেশি ধারণক্ষমতার Hard Disk ব্যবহার করা হয়।

💡 জানেন কি?

  • 🌍 বিশ্বের প্রথম Hard Disk-এর ধারণক্ষমতা ছিল মাত্র 5 MB।
  • 💾 বর্তমানে একটি HDD-তে 20TB বা তারও বেশি ডেটা রাখা যায়।
  • 🎬 বড় বড় সিনেমা ও ভিডিও এডিটিং কোম্পানিতে এখনও HDD ব্যবহার করা হয়।
  • 📦 Backup Storage-এর জন্য HDD এখনও খুব জনপ্রিয়।

❓ Frequently Asked Questions (FAQ)

১. Hard Disk কী?

Hard Disk হলো একটি স্থায়ী (Non-Volatile) Storage Device, যেখানে কম্পিউটারের সব তথ্য সংরক্ষণ করা হয়।

২. HDD-এর পূর্ণরূপ কী?

HDD-এর পূর্ণরূপ হলো Hard Disk Drive।

৩. কম্পিউটার বন্ধ করলে Hard Disk-এর তথ্য কি মুছে যায়?

না। Hard Disk-এর তথ্য স্থায়ীভাবে সংরক্ষিত থাকে।

৪. HDD এবং SSD-এর মধ্যে কোনটি দ্রুত?

SSD অনেক দ্রুত, কারণ এতে কোনো চলমান যন্ত্রাংশ নেই।

৫. Hard Disk-এর ধারণক্ষমতা কত হতে পারে?

বর্তমানে 500GB, 1TB, 2TB, 4TB, 8TB, 10TB এমনকি 20TB-এরও বেশি ধারণক্ষমতার HDD পাওয়া যায়।

৬. External Hard Disk কী?

USB-এর মাধ্যমে কম্পিউটারের বাইরে থেকে সংযুক্ত করা যায় এমন Hard Disk-কে External Hard Disk বলে।

৭. Hard Disk নষ্ট হলে কী হবে?

সংরক্ষিত তথ্য হারিয়ে যেতে পারে। তাই নিয়মিত Backup রাখা উচিত।

🎯 উপসংহার

Hard Disk কম্পিউটারের অন্যতম গুরুত্বপূর্ণ Storage Device। এটি দীর্ঘ সময় ধরে তথ্য নিরাপদে সংরক্ষণ করে এবং Operating System, Software ও ব্যক্তিগত ফাইল রাখার জন্য ব্যবহৃত হয়। যদিও SSD-এর তুলনায় এটি ধীরগতির, তবুও কম খরচে বেশি স্টোরেজ পাওয়ার জন্য HDD এখনও ব্যাপকভাবে ব্যবহৃত হয়।
